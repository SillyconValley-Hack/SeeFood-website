The Product

SeeFood is a mobile application that lets you detect what kind of food is presented in front of you. It currently supports hotdog detection and maybe only hotdog forever.

Website

The website is built on top of Sapper. Please take a look at Sapper .

Mobile Application

The mobile application is built with Flutter. Check out flutter.dev .

Backend

We have a Django server running as an API server. For image recognition, we have one built with Pytorch and the other using Google's search by image .

Get it and eat some hotdogs

SeeFood is currently under development. Feel free to join us on GitHub.
